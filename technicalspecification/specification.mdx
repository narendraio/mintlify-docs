title: "Specifications for Compatibility, if Applicable"
---
### Website Platform Compatibility

Heatmaphero is compatible with various web-based platforms, irrespective of the technology used (HTML, JavaScript, etc.). 

The platform does not discriminate based on the content management system (CMS) or the programming language used to build the website.

### Browser Compatibility

Heatmaphero is compatible with the following browsers:
1. Chrome
2. Firefox
3. Safari
4. Microsoft Edge
5. Opera

Other modern and widely used browsers

### Mobile Compatibility
Heatmaphero is compatible with mobile browsers on devices running iOS, Android, and other major operating systems.
