---
title: "Initial Configuration Steps"
---
### Dashboard Setup: 
Upon logging into the Heatmaphero dashboard navigate through the intuitive interface to access various features like heatmaps, session recordings, and alerts.

Set up your preferences, such as frequency of data collection and types of alerts to receive.

### Customization and Filters:

Tailor the settings to your specific needs, such as excluding specific pages or user actions from data collection.

Apply filters to focus on particular user segments or behavior patterns.

### Code Validation:
Ensure the code snippet integration is correctly implemented across all web pages.

Verify the correct functioning of Heatmaphero's features by viewing initial data and recordings.
