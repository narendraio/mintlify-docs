---
title: "Its Purpose and Intended Audience"
---

**Purpose**

HeatmapHero's developer documentation serves as a comprehensive guide for integrating and leveraging the features of HeatmapHero's user behavior analytics platform within applications or websites.
</br >
**Intended Audience** 

Developers, engineers, and technical teams responsible for integrating, optimizing, and customizing user behavior analytics tools within web-based applications. This includes, but is not limited to, front-end developers, UX/UI designers, product managers, and those responsible for optimizing website performance.
